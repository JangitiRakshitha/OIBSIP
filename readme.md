# Machine Learning & Data Analysis Projects

## 🚗 Project 1: Car Price Prediction

### 📌 Project Overview
Car Price Prediction is a machine learning project that predicts the selling price of a car based on factors such as brand, year, mileage, fuel type, transmission, and ownership details.

### 🎯 Objectives
- Analyze factors affecting car prices  
- Build a regression model to predict car prices  
- Improve decision-making for buyers and sellers  

### ✨ Features
- Data preprocessing and cleaning  
- Feature encoding  
- Regression-based price prediction  
- Model performance evaluation  

### 🛠️ Technologies and Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn

## 📧 Project 2: Email Spam Detection

### 📌 Project Overview
Spam Detection is a machine learning classification project that identifies whether an email is spam or not spam (ham) using text analysis and NLP techniques.

### 🎯 Objectives
- Detect spam emails automatically  
- Apply Natural Language Processing techniques  
- Build an accurate classification model  

### ✨ Features
- Text preprocessing and cleaning  
- Feature extraction using vectorization  
- Spam vs Ham classification  
- Model evaluation using accuracy metrics  

### 🛠️ Technologies and Tools Used
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Natural Language Processing (NLP)

## 📉 Project 3: Unemployment Analysis

### 📌 Project Overview
Unemployment Analysis focuses on studying unemployment trends across regions and time periods, especially during the COVID-19 pandemic, using data analysis and visualization techniques.

### 🎯 Objectives
- Analyze unemployment rates over time  
- Understand the impact of COVID-19 on employment  
- Perform region-wise unemployment analysis

- Exploratory Data Analysis (EDA)  
- Trend and comparative analysis

### 🛠️ Technologies and Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn
